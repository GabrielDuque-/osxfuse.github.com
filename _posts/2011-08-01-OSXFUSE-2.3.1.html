--- layout: post title: Release of OSXFUSE 2.3.1 ---

New Features

  • Enhanced MacFUSE compatibility mode to make OSXFUSE compatible with the latest TrueCrypt release.

Fixes

  • Fixed a bug in build script target reload.
  • Fixed a bug in OSXFUSE.framework that may have resulted in the selector removeExtendedAttribute:ofItemAtPath:error: never being called. Tracked as issue #2: "removeExtendedAttribute never called". Reported and fixed by Dan Cooper.
  • Fixed bundle identifier of the OSXFUSE kernel extension.

Known Issues

  • Many simultaneous file system operations on an OSXFUSE/MacFUSE volume may trigger a file system deadlock, that renders the file system being unresponsive and therefore being ejected. This bug affects solely Macs running a 64 bit kernel.
